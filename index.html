🔑 Подключение к GitHub

Для автоматического сохранения данных нужен GitHub токен. Это делается один раз.

1. Перейди по ссылке: github.com → Settings → Tokens
2. Нажми Generate token
3. Скопируй токен (начинается с ghp_)
4. Вставь сюда и нажми «Подключить»
